# Threads 投稿テンプレ（不動産AI伴走）

導線：Threads → プロフィールリンク → LP → AI診断 or LINE登録

---

## 投稿1：問題提起型（共感狙い）

```
「うちの規模でAIなんて関係ない」
そう言ってた不動産会社の社長が、
反響対応に毎日3時間取られてた話。

スタッフ4人で月50件の反響を捌くって、もう人の限界です。

AI入れる前に、まず"どこが詰まってるか"を可視化するのが先。
不動産特化のAI診断、無料で作りました。
60秒・10問。プロフのリンクから。
```

## 投稿2：HOW型（具体ノウハウ）

```
不動産会社がAIで真っ先に楽になるのは「物件募集文の作成」です。

理由：
・毎回ゼロから書いてる
・担当ごとに品質差がある
・ポータル×SNSで使い回しが効かない

ChatGPTに物件情報を渡すだけで、SUUMO用／LINE用／Instagram用が一発で出ます。
プロンプトはプロフのリンクから配布中。
```

## 投稿3：体験談型（一人称）

```
個人で不動産会社のAI支援を始めて、最初に驚いたこと。

「うちの業務、ぜんぶ言語化できてないんですよ」

そう。多くの会社で詰まってるのはAIじゃなくて、業務の見える化のほう。

だから僕は最初に必ず「業務診断」をやります。
そこを飛ばして導入すると、ツールが机の上で死ぬので。
```

## 投稿4：質問投げかけ型（コメント誘発）

```
不動産会社のみなさんに聞きたいです。

1日のうちで「これAIに任せたい…」って思う業務、なんですか？

・反響メール返信
・物件募集文の作成
・追客LINE
・契約書チェック
・SUUMO更新

僕の感覚だと、コスパ最強は2番目です。理由はリプで。
```

## 投稿5：診断結果ティーザー

```
今日診断を受けてくれた不動産会社さん（社員8名）の結果。

AI導入準備度：42点
業務効率化の伸びしろ：87点
即効性：91点

つまり、「まだ何もやってない」けど「やれば一気に変わる」状態。
これ、けっこう多いパターンです。

無料診断、60秒で出ます。プロフから。
```

---

## 運用ルール

- **投稿時間**：平日 8:00 / 12:00 / 19:00 のいずれか
- **頻度**：週3〜5本（連投1日2本までに留める）
- **最初の15分**：自分のコメント1つ・別アカウントから「？」リプで火種を作る
- **CTA**：すべて「プロフのリンクから」に統一（DM・コメント誘導しない）
- **絵文字**：原則使わない。ビジネス寄りで信頼感を優先
